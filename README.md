# Artificial Intelligence Cosmos

## Overview
Artificial Intelligence Cosmos is a superior AI services company specializing in advanced neural networks, MLOps, prompt engineering, and compliance engineering.

## Setup & Viewing

This project is built using modern HTML5, CSS3, and Vanilla JavaScript. No compilation or installation is required.

### How to View
1.  Navigate to the `d:\Git\1` directory in your file explorer.
2.  Double-click on `index.html` to open it in your default web browser.
3.  Navigate through the site using the menu links.
